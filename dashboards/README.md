# Monitoring Dashboards

This project uses three dashboards for grafana for monitoring purposes. I did not create these dashboards and take no credit for them. They each belong to their respective owners. The third dashboard was initially designed to work on red hat systems. Since this project uses an Ubuntu system some modifications were made to the expressions in the yaml file in order to be compatible with ubuntu sshd logs. The ssh dashboard included in this repo is the modified version designed for ubuntu systems.
